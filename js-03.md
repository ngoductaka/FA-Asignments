# this & Object prototype
    1. this:
    -this là một trong những cơ chế gây rối nhất trong JS, theo em this là gì ?
    -Cách hiểu 1: this trỏ tới function f, đúng hay sai?

        cách 1: 
        
        ```
        
        function f() {
            console.log(this.a);
        }
        f();// log ra "undefined" vì:
        // this trong trường hợp này là globle object 
        // trong global chưa định nghia thuộc tính a;
        //cách 2:
        var o = {
            a: "dnd",
            method: fn
        }
        o.method();// log ra "dnd" vì :
        // this đại diện cho object o
        ```
